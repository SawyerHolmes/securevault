About.

Version 0.10.0 · Built 2026-05-26

What it is

A local-first password manager that lives entirely in your browser. Vault data is encrypted on the device with AES-256-GCM under a key derived from your master password via PBKDF2-HMAC-SHA-256 at 600,000 iterations. Nothing is uploaded unless you turn on Gist sync — and even then the only thing that leaves is the encrypted blob.

Security model

  • AES-256-GCM, 96-bit IV per record, authenticated.
  • PBKDF2-HMAC-SHA-256, 600 000 iterations, per-vault random salt.
  • Master password never leaves the device. The derived key is held in sessionStorage and cleared on lock.
  • GitHub PAT (when sync is configured) is also encrypted with the vault key.
  • Breach check uses Have I Been Pwned k-anonymity — only the first 5 hex chars of the SHA-1 hash hit the network.
  • Exponential brute-force lockout: 30 s, 1 m, 2 m, 4 m, 8 m, 16 m, 32 m, 1 h cap.

Built with

Vanilla HTML / CSS / JS. No framework, no build step, no dependency on a remote API except (optionally) the GitHub Gist endpoint for sync and the Have I Been Pwned range API for breach checks. Inter is served locally; the ten icons in use are inlined from Lucide.
